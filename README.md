## üöÄ Core Features

- **Authentication & Authorization** ‚Äì Secure JWT-based authentication with Passport.js and role-based access control.
- **Referral System** ‚Äì Complete referral code management with hierarchical relationships.
- **Transaction Management** ‚Äì Secure transaction workflows with admin approval.
- **Queue System** ‚Äì BullMQ-powered background processing for async tasks and email jobs.
- **Real-time Notifications** ‚Äì Instant alerts for transaction approvals and key events.
- **Automated Reporting** ‚Äì Scheduled cron jobs generating financial summaries.
- **Email Service** ‚Äì Template-based transactional email system integrated with Nodemailer.
